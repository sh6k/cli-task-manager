# Advanced C++ CLI Task Manager

A high-performance command-line task management system built to demonstrate advanced Data Structures and Algorithms in C++.

## 🚀 Key Engineering Features
* **Dynamic Data Management:** Implemented a custom **Singly Linked List** for efficient memory allocation of tasks.
* **Algorithmic Optimization:** Integrated a recursive **Merge Sort** algorithm to organize tasks by priority with $O(n \log n)$ time complexity.
* **Memory Safety:** Managed heap memory manually with a custom class destructor to ensure zero memory leaks.
* **Data Persistence:** Developed a file I/O system for saving and loading task states across sessions.

## 🛠️ Built With
* C++17
* Visual Studio 2022
* Git/GitHub

## 🖥️ How to Run
1. Clone the repository: `git clone https://github.com/sh6k/cli-task-manager.git`
2. Open the `.sln` file in Visual Studio.
3. Press `Ctrl + F5` to compile and run.
